# Lycolia's zsh configure files

## Enviroments Support

-   MSYS2/zsh
-   Ubuntu

## Usage

-   download this repository by zip
-   extract to `~/`

### for MSYS2

Install msys2/git for using prompt and rename so will being not conflict with Git for Windows

```sh
pacman -S git
mv /usr/bin/git /usr/bin/msys2_git
```
